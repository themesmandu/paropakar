Paropakar, a WordPress theme is mainly for charity and donation based websites. With custom design for front page, users can design as per their requirement. Includes header image slider, footer client logo slider and a template for client testimony.

== Description ==

Paropakar theme brings your site to life with header slider and immersive featured images. With a focus on charity sites, it features multiple sections on the front page as well as widgets, navigation and social menus, a logo, and more. Personalize the theme with very easy and user-friendly customizer. Users can make the front page as per their requirement. Includes header image slider, footer client logo slider and a template for client testimony.

For more information about Paropakar theme please go to https://www.themesmandu.com/product/paropakar/.

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Paropakar theme in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your paropakar theme right away.
4. Go to http://merchrest.com/paropakar/ for a guide on how to customize this theme.
5. Navigate to Appearance > Customize in your admin panel and customize to taste.
